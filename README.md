# Freelance · Веб-разработка — материалы и расценки

Комплект маркетинговых материалов для услуг веб-разработки в Кыргызстане:
лендинг, PDF-прайс, тексты и баннер для Telegram.

Цены везде указаны в **сомах (KGS) + USD** по курсу **~89 сом/$**.

## 🔄 Перерендерить PDF и PNG после правок

Источники — `.html`. После изменений пересобрать через headless Chrome:

```bash
cd ~/projects/personal/freelance
CHROME="/Applications/Google Chrome.app/Contents/MacOS/Google Chrome"

# Прайс → PDF
"$CHROME" --headless --no-pdf-header-footer \
  --print-to-pdf="Прайс-лист.pdf" "file://$PWD/Прайс-лист.html"

# Баннер → PNG (1080×1080)
"$CHROME" --headless --window-size=1080,1080 \
  --screenshot="Telegram-баннер.png" "file://$PWD/Telegram-баннер.html"
```
